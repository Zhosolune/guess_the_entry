# 需求文档 — UI 极简风格与统一间距改造

## 介绍
本需求旨在将当前界面调整为极简风格，统一主要区域间距与对齐方式，移除紫色主题与多余装饰，采用中性色与扁平化设计，并通过 CSS 变量管理主题色，确保响应式与跨浏览器一致性。

## 需求

### 需求 1 — 三大区域间距与对齐统一

**用户故事：** 作为玩家，我希望词条文本区域、已猜对字符区域和坟场区域在间距和对齐上保持一致，以便信息层级更清晰，阅读更舒适。

#### 验收标准（EARS）
1. While 游戏主页面呈现时, when 渲染“文本区域/已猜对字符/坟场”三个模块, the 系统 shall 统一三个模块的内边距（padding）为相同数值（以“已猜对字符区域”为基准），并在主容器内保持左右对齐。
2. While 视口尺寸变化（≥640px 与 <640px）时, when 响应式布局触发, the 系统 shall 保持三个模块的统一间距与对齐不变，避免出现错位或过密。
3. While 浏览器缩放或系统字体变化时, when 布局重新计算, the 系统 shall 不导致三个模块间距不一致或出现溢出。

### 需求 2 — 极简风格主题替换

**用户故事：** 作为玩家，我希望整体界面采用中性色系与简洁样式，以避免视觉干扰、提升信息可读性。

#### 验收标准（EARS）
1. While 页面加载时, when 应用主题样式, the 系统 shall 移除当前紫色主题与渐变背景，采用中性色（灰/黑/白）为主色；主卡片背景改为白底或近白底。
2. While 样式渲染时, when 渲染卡片容器, the 系统 shall 取消阴影（box-shadow），改为 `1px` 浅色边框（建议 `#e0e0e0`）。
3. While 渲染按钮时, when 用户看到交互控件, the 系统 shall 使用扁平化按钮样式（无立体阴影/光效），并保留 hover/focus 的轻微色阶变化与边框强调。
4. While UI 元素渲染时, when 发现纯装饰性元素（扫光、强渐变、复杂阴影）存在, the 系统 shall 移除或简化，保留必要的功能性标识。

### 需求 3 — 保留元素与样式简化

**用户故事：** 作为玩家，我需要保留图标与标题等功能性标识，但希望样式更简洁。

#### 验收标准（EARS）
1. While 渲染三个主要区域, when 显示区块图标, the 系统 shall 保留既有图标样式，不更改图标形状与颜色主题，仅调整其周围留白使其与文本对齐。
2. While 渲染标题文本, when 显示模块标题, the 系统 shall 保留标题文字但简化样式（常规字重/中等字号/去除额外装饰线与强渐变）。

### 需求 4 — 主题与过渡实现要求

**用户故事：** 作为开发维护者，我希望使用 CSS 变量管理主题色，统一过渡动效并保持响应式一致。

#### 验收标准（EARS）
1. While 页面初始化, when 注入主题变量, the 系统 shall 在 `:root` 定义主题 CSS 变量（中性色主色、边框色、文本色、按钮色阶等），并在暗色模式下提供覆盖。
2. While 元素状态变化, when 触发样式过渡, the 系统 shall 使用 `transition: all 0.3s ease`（或等效细分）作为统一动效策略，过渡范围包含背景色、边框色、间距变化不产生跳动。
3. While 不同屏幕尺寸下, when 响应式规则生效, the 系统 shall 维持统一视觉效果（间距、边框、背景、按钮样式），不因断点切换出现突兀样式差异。

### 需求 5 — 跨浏览器一致性

**用户故事：** 作为玩家，我希望在主流浏览器中看到一致的视觉呈现。

#### 验收标准（EARS）
1. While 使用 Chrome/Firefox/Safari 浏览器, when 加载页面, the 系统 shall 保持样式一致，无明显色差、阴影或边框呈现问题；按钮交互状态（hover/focus/disabled）表现一致。
2. While 浏览器缩放与系统暗色模式切换, when 主题变量与暗色覆盖生效, the 系统 shall 样式稳定，不出现闪烁或布局错乱。

## 范围与约束
- 仅前端样式与主题实现，避免不必要的组件结构变更；坚持最小可用实现（YAGNI/KISS）。
- 保持既有功能不变：输入交互、揭示动画、坟场与已猜对字符面板功能保持。
- 不引入大型 UI 框架或复杂主题系统，使用现有 Tailwind/CSS 结构与轻量变量实现。

## 验收标准汇总
- 整体界面干净简洁，无多余装饰；三大区域间距统一、视觉层次分明；保留的功能性元素清晰可见。
- 响应式布局与暗色模式下表现一致；在 Chrome/Firefox/Safari 上显示一致。
